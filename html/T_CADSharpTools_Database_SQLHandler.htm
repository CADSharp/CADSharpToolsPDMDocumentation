Remarks
To use the SQLHandler class:
  • Set the ConnectionString property.
  • Invoke Open to connect to the database.
  • Perform your SQL transaction by invoking PerformSQLTransaction(FuncSqlConnection, MethodReturnBoolean) if you don't require a return or PerformSQLTransactionT(FuncSqlConnection, MethodReturnT) if you require a return value.
  • Close connection when done. You may reinvoke Open to reopen the connection.
  • Dispose when application is exiting.
